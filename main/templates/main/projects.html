Timetable Telegram bot

Timetable telegram bot - is a telegram bot created for quick and easy viewing of the schedule of classes at the URFU University for the FT-570101 group. The bot works in 3 stages:

  1. First, the bot takes the user's request, which is formed using a special keyboard
  2. Then the Bot turns to the official website of the university, parses the necessary schedule data for the group
  3. Then he transfers information to the user

Google Chrome extention - notes

Google Chrome расширение (notes) - is an MVP extension that allows you to work with notes:

  1. Add
  2. Delete
  3. Mark as done

The extension is written in pure JavaScript and works with Google Chrome's local storage to record user input.
