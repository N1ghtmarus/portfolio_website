Телеграм бот

Телеграмм бот для просмотра расписания - это бот, созданный для быстрого и удобного просмотра расписания занятий в университете УрФУ для группы ФТ-570101. Бот работает в 3 этапа:

  1. Сначала бот принимает запрос пользователя, который формируется с помощью специальной клавиатуры в телеграме
  2. Далее Бот обращается на официальный сайт вуза и парсит необходимые данные расписания для группы
  3. Затем он передает информацию пользователю
